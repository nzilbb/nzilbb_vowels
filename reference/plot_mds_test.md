# Plot [`mds_test()`](https://nzilbb.github.io/nzilbb_vowels/reference/mds_test.md) results

**\[experimental\]** Plot output from
[`mds_test()`](https://nzilbb.github.io/nzilbb_vowels/reference/mds_test.md).

## Usage

``` r
plot_mds_test(mds_test)
```

## Arguments

- mds_test:

  Object of class `mds_test_results` (generated by
  [`mds_test()`](https://nzilbb.github.io/nzilbb_vowels/reference/mds_test.md)).

## Value

`ggplot` object.

## Examples

``` r
mds_result <- mds_test(
    sim_matrix,
    n_boots = 10,
    n_perms = 10,
    test_dimensions = 3,
    mds_type = 'interval'
 )
 plot_mds_test(mds_result)
```
